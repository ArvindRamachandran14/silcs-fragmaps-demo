# M5.5a Preview Index - Reset Defaults Semantics Refinement

## M5.5a Scope Lock
- `M5.5a` only.
- Includes `Reset defaults` semantics refinement only:
  - reset per-map iso values to canonical defaults,
  - keep map visibility unchanged.
- Excludes:
  - changes to `Hide all` semantics,
  - reliability/retry UX changes (remain `M5.6`),
  - any ligand/protein/tab-layout behavior changes.

## Coverage Checklist
- [x] `Default` state: mixed map visibility with non-default iso values before reset.
- [x] `Loading` state: in-progress `Reset defaults` disables only the reset action while operation is active.
- [x] `Empty` state: when no maps are visible, reset still normalizes iso defaults without introducing visible maps.
- [x] `Error` state: failed iso-reset path isolates the affected row and keeps visibility unchanged.
- [x] `Success` state: iso values return to defaults while current visibility remains exactly as before reset.
- [x] Scope constrained to `M5.5a` only.

# M4 Ligand Workflow Design Preview Packet

This packet covers the M4 Design Preview Gate in phased scope (`M4A`, `M4B`; `M4C` deferred).

## Gate Status
- Active milestone preview track: `M4B`
- Gate state: `APPROVED UI PREVIEW`
- Required approval token: `APPROVED UI PREVIEW`

## Scope
In scope for this packet:
- `M4A` single-ligand core workflow previews (archived in this packet).
- `M4B` featured-ligand expansion previews for desktop switching states.

Out of scope for this packet:
- `M4C` full searchable ligand list and deterministic ordering.
- M5+ controls/workflows.

## Packet Structure
- `preview-index.md`: M4A preview checklist and traceability.
- `m4b-preview-index.md`: M4B preview checklist and traceability.
- `approval-log.md`: approval record and reviewer comments.
- `desktop/`: desktop preview artifacts for M4A + M4B states.
- `states/`: per-state guidance and artifact links.

## Rendering Policy
- Desktop artifacts are required for M4A and M4B preview approval.
- Mobile artifacts are intentionally omitted for this scope per current reviewer instruction.
